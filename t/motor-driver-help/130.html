If you use the 32Amp version of Sabertooth you can actually program the Sabertooth itself … independant of the Arduino … and limit the number of amps it will allow. If it was me, and knowing how expensive those suckers are to replace, I’d build an in-line 30A fuse into the power line going to each motor. Fuses are a lot cheaper to replace
